# Файлы
## Main.py - основный файл тренировки и предсказания нейросетей
## neuronv2.py - отвечает за сегментации и выделления дефектов изображения на видеопотоке с камеры в прямом эфире
## OpencvReader.py - Выравнивает и кадрирует видеопоток для передачи кадра в обработку нейросети
## OpenCV.py - файл раскадровки и разделения изображения
